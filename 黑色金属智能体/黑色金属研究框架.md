## 二、黑色金属分类

### 2.1 钢铁产品分类

| 类型 | 主要品种 | 下游应用 | 特点 |
|-----|---------|---------|------|
| **长材** | 螺纹钢、线材 | 建筑、基建 | 周期性强，地产主导 |
| **板材** | 热轧、冷轧、中厚板 | 汽车、家电、机械 | 制造业属性强 |
| **特钢** | 不锈钢、轴承钢、齿轮钢 | 高端制造 | 技术壁垒高，盈利稳定 |

### 2.2 原料分类

| 原料 | 作用 | 定价特点 |
|-----|------|---------|
| **铁矿石** | 铁元素来源 | 国际定价，四大矿主导 |
| **焦炭/焦煤** | 还原剂+燃料 | 国内定价，供给约束 |
| **废钢** | 电炉原料 | 国内定价，资源属性 |

## 三、产业链结构

### 3.1 长流程（高炉-转炉）

```
上游          中游              下游
铁矿石  →  高炉炼铁  →  转炉炼钢  →  连铸  →  轧材  →  应用
焦炭         ↓           ↓
焦煤       生铁        粗钢
废钢(10-30%)
```

**成本构成**：
- 铁矿石：40-50%
- 焦炭/焦煤：20-30%
- 废钢：10-20%
- 其他：10-20%

### 3.2 短流程（电炉）

```
废钢  →  电炉炼钢  →  连铸  →  轧材  →  应用
```

**特点**：
- 投资小、建设快
- 灵活性高，可快速启停
- 碳排放低，符合双碳目标
- 成本受废钢价格和电价影响

### 3.3 铁矿石产业链

```
海外矿山  →  航运  →  港口  →  钢厂
(四大矿)    (海岬型)  (青岛/日照)
```

**四大矿**：
- 淡水河谷（巴西）
- 力拓（澳洲）
- 必和必拓（澳洲）
- FMG（澳洲）

## 四、供需平衡分析

### 4.1 钢铁供需平衡框架

```
供给端              需求端              平衡
─────────────────────────────────────────────
粗钢产量    +    地产用钢
进口钢材    +    基建用钢
            -    制造业用钢
            -    出口钢材
─────────────────────────────────────────────
           =  供需缺口
```

### 4.2 需求结构

| 下游 | 占比 | 驱动因素 |
|-----|------|---------|
| 地产 | 30-35% | 新开工、施工面积 |
| 基建 | 20-25% | 专项债、财政支出 |
| 机械 | 15-20% | 工业投资、出口 |
| 汽车 | 5-10% | 汽车产量 |
| 家电 | 3-5% | 地产后周期 |
| 其他 | 10-15% | 船舶、集装箱等 |

### 4.3 关键跟踪指标

| 指标 | 含义 | 数据频率 |
|-----|------|---------|
| 高炉开工率 | 产能利用率 | 周 |
| 螺纹钢库存 | 社会库存+钢厂库存 | 周 |
| 铁矿石港口库存 | 港口现货库存 | 周 |
| 成材成交量 | 终端需求强度 | 日 |
| PMI | 制造业景气度 | 月 |

## 五、价格影响因素

### 5.1 钢价驱动框架

```
长期因素（1-3年）      中期因素（3-12月）    短期因素（1-3月）
─────────────────────────────────────────────────────
地产周期    ←→    高炉开工率    ←→    库存变化
产能政策          原料成本            市场情绪
碳中和            季节性因素          期货持仓
```

### 5.2 铁矿石价格驱动

```
供给端                需求端
─────────────────────────────────────────────
四大矿产量    ←→    中国生铁产量
非主流矿          钢厂利润
港口库存          补库需求
```

**关键结论**：
- 铁矿石定价权在海外，中国是价格接受者
- 钢厂利润决定铁矿价格上限
- 港口库存反映供需紧张程度

### 5.3 利润分配

```
产业链利润分配（历史变化）：

2016-2018年：钢铁占产业链利润 46%
2019-2020年：钢铁占产业链利润 37%
2021-至今：钢铁占产业链利润 <10%

原因：铁矿、焦煤供给更紧缺，议价能力更强
```

## 六、库存周期

### 6.1 四阶段模型

| 阶段 | 需求 | 库存 | 价格 | 投资策略 |
|-----|------|------|------|---------|
| **被动补库** | ↓ | ↑ | ↓ | 规避风险 |
| **主动去库** | ↓ | ↓ | 筑底 | 左侧布局 |
| **被动去库** | ↑ | ↓ | ↑ | 积极配置 |
| **主动补库** | ↑ | ↑ | 见顶 | 持有为主 |

### 6.2 季节性特征

```
Q1（春节前后）：累库期，需求淡季
Q2（3-5月）：去库期，需求旺季
Q3（6-8月）：震荡期，高温淡季
Q4（9-11月）：去库期，金九银十
```

## 九、智能体功能设计

### 9.1 核心功能

1. **行业分析**
   - 输入：品种名称（螺纹钢/热卷/铁矿石）
   - 输出：行业定义、产业链、关键指标

2. **供需分析**
   - 输入：品种类型
   - 输出：供需平衡框架、关键跟踪指标

3. **成本分析**
   - 输入：钢厂类型（长流程/短流程）
   - 输出：成本构成、利润测算

4. **周期判断**
   - 输入：库存趋势、需求趋势
   - 输出：周期阶段、价格展望、投资建议

5. **公司筛选**
   - 输入：排序指标
   - 输出：公司列表、排序结果

6. **API查询生成**
   - 输入：数据类型、证券代码
   - 输出：Choice API查询语句
