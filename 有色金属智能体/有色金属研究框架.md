## 二、有色金属分类

### 2.1 工业金属

| 金属 | 主要用途 | 定价权 | 关键指标 |
|-----|---------|--------|---------|
| **铜** | 电线电缆、建筑、汽车 | 全球定价(LME) | TC/RC、库存、升贴水 |
| **电解铝** | 建筑、交通、电力 | 中国定价主导 | 产能利用率、氧化铝价格 |
| **锌** | 镀锌、合金 | 全球定价 | 加工费、库存 |
| **镍** | 不锈钢、电池 | 全球定价 | NPI价格、库存 |

### 2.2 贵金属

| 金属 | 主要用途 | 定价逻辑 |
|-----|---------|---------|
| **黄金** | 珠宝、投资、央行储备 | 实际利率 = 名义利率 - 通胀 |
| **白银** | 工业用途、投资 | 金银比 + 工业需求 |

### 2.3 能源金属

| 金属 | 主要用途 | 特点 |
|-----|---------|------|
| **锂** | 动力电池、储能 | 盐湖/锂辉石/锂云母三种来源 |
| **钴** | 电池正极材料 | 刚果主导供给 |
| **镍** | 电池高镍化 | 从不锈钢向电池转型 |

## 三、产业链结构

### 3.1 铜产业链

```
上游          中游              下游
铜矿开采  →  铜精矿  →  粗铜  →  精炼铜  →  铜材  →  应用
   ↓                           ↓            ↓
紫金矿业                     江西铜业      精达股份
洛阳钼业                     铜陵有色      海亮股份
```

**关键环节**：
- TC/RC加工费：反映矿端与冶炼端的供需关系
- 废铜：补充供给，价差影响替代

### 3.2 电解铝产业链

```
铝土矿  →  氧化铝  →  电解铝  →  铝材  →  应用
   ↓         ↓          ↓         ↓
进口为主   中国铝业    中国宏桥    南山铝业
          南山铝业    云铝股份    明泰铝业
                     神火股份
```

**成本构成**：
- 氧化铝：40%
- 电力：40%
- 阳极：10%
- 其他：10%

**政策约束**：
- 产能天花板：4500万吨
- 碳中和约束：高能耗行业

### 3.3 锂产业链

```
资源端        中游材料         下游应用
锂辉石   →   碳酸锂   →    正极材料  →  电池
盐湖         氢氧化锂
锂云母
   ↓            ↓              ↓
赣锋锂业     容百科技       宁德时代
天齐锂业     当升科技       比亚迪
盐湖股份     德方纳米       亿纬锂能
```

## 四、供需平衡分析

### 4.1 供需平衡表框架

```
供给端              需求端              平衡
─────────────────────────────────────────────
矿产产量    +    下游消费
再生供给    +    出口需求
进口量      -    库存变化
─────────────────────────────────────────────
           =  供需缺口
```

### 4.2 关键跟踪指标

| 指标 | 含义 | 数据频率 |
|-----|------|---------|
| 社会库存 | 流通领域库存 | 周/月 |
| 交易所库存 | LME/SHFE显性库存 | 日 |
| 加工费 | TC/RC，供需信号 | 周 |
| 升贴水 | 现货-期货价差 | 日 |
| 进出口量 | 海关数据 | 月 |

## 五、库存周期

### 5.1 四阶段模型

| 阶段 | 需求 | 库存 | 价格 | 投资策略 |
|-----|------|------|------|---------|
| **被动补库** | ↓ | ↑ | ↓ | 规避风险 |
| **主动去库** | ↓ | ↓ | 筑底 | 左侧布局 |
| **被动去库** | ↑ | ↓ | ↑ | 积极配置 |
| **主动补库** | ↑ | ↑ | 见顶 | 持有为主 |

### 5.2 判断方法

```
1. 观察库存趋势（周度/月度）
2. 判断需求变化（下游开工率、订单）
3. 结合价格走势验证
```

## 六、价格影响因素

### 6.1 铜价驱动框架

```
长期因素（1-3年）      中期因素（3-12月）    短期因素（1-3月）
─────────────────────────────────────────────────────
全球经济增长    ←→    TC/RC加工费    ←→    美元指数
铜矿资本开支          库存水平            市场情绪
绿色转型需求          中国需求            投机持仓
```

### 6.2 铝价驱动框架

```
长期因素               中期因素            短期因素
─────────────────────────────────────────────────────
产能天花板(4500万吨)   氧化铝价格          社会库存
碳中和政策            电力成本            下游开工率
需求结构变化          产能利用率          季节性因素
```

### 6.3 金价驱动框架

```
实际利率 = 名义利率 - 通胀预期

金价与美国实际利率负相关

影响因素：
├── 美联储货币政策
├── 通胀预期
├── 美元指数
├── ETF持仓
└── 央行购金
```
