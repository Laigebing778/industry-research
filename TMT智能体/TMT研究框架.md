### 2.4 技术演进路线

**核心技术方向**：大带宽、低功耗、低成本

```
器件/材料层面改进（小改进）：
├─ EML：速率从100G/lane升级到200G/lane
├─ VCSEL：短距离采用高速VCSEL降低功耗成本
├─ TFLN：薄膜铌酸锂调制，带宽更高、功耗更低
└─ DSP/CDR：集成TIA/DRV，省略或集成降低功耗

结构层面改进（大改进）：
├─ 硅光：多通道集成，甚至与部分电芯片集成
├─ LPO：省略DSP/CDR，线性驱动降功耗（20-30% BOM下降）
└─ CPO：光电共封装，提高集成度，减少电信号传输距离
```

**硅光技术对比**：

| 维度 | 硅光方案 | 传统InP方案 |
|-----|---------|------------|
| 晶圆尺寸 | 8-12英寸 | 2-3英寸 |
| 工艺精度 | 65nm-250nm | 0.3-0.5um |
| 集成度 | 耦合器、波导、调制器、探测器单片集成 | 难实现高密度集成 |
| 芯片良率 | >80% | <40% |
| 成本 | 量大可很低 | 受限于良率和固定开支 |

**2025年硅光上量核心原因**：
- 良率高：硅光芯片良率>70%，200G EML良率从30-40%提升需时间
- 快速交付：满足AI算力爆发需求
- 降本需求：旭创2025年预计60-70%采用硅光

## 三、通信行业研究框架

### 3.1 产业链结构

```
上游：光芯片/光纤/光缆/电子元器件/PCB
中游：通信主设备商（华为/中兴/爱立信/诺基亚）
下游：电信运营商（中国移动/中国电信/中国联通）
终端：终端设备/通信设备商/物联网模组
应用：内容服务商/应用服务提供商
```

### 3.2 核心逻辑：运营商Capex/OPEX是行业晴雨表

- **Capex（资本开支）**：影响设备商、光纤光缆、PCB等上游行业

- **关键判断**：运营商Capex有周期性，影响行业投资节奏

### 4.4 核心指标体系

| 维度 | 核心指标 | 定义 | 业务意义 |
|-----|---------|-----|---------|
| **收入** | ARR（年度经常性收入） | 经常性软件收入的年化金额 | 衡量未来收入潜力 |
| | NRR（净收入留存率） | (期末收入-新增客户收入)/期初收入×100% | NRR>100%表示现有客户贡献增长 |
| **客户** | CAC（客户获取成本） | 销售与营销费用/新增客户数 | 越低表示资源投入产出比越高 |
| | LTV（客户生命周期价值） | 客户平均收入×平均客户生命周期 | 评估客户长期贡献 |
| **效率** | CAC回收期 | CAC/(客户年均收入-服务成本) | 越短表示资金周转效率越高 |
| **盈利** | 毛利率 | (收入-服务成本)/收入×100% | SaaS模式通常具有较高毛利率 |

### 4.5 SaaS vs 传统软件对比

| 维度 | 传统软件公司 | SaaS公司 |
|-----|------------|---------|
| **收入来源** | 一次性软件授权销售 | 订阅制服务费用（按年/月） |
| **收入确认** | 交易完成时一次性确认 | 客户生命周期内逐步确认 |
| **核心增长驱动** | 新客户获取 | 客户留存与扩展（续约、增购） |
| **客户门槛** | 较高 | 较低（订阅制降低初始投入） |

### 4.6 收入公式

```
收益 = 客户数量 × 客户平均生命周期价值（LTV）

其中：
- 客户数量 = 潜在客户获取 × 转化率
- LTV = ARPU × 客户持续使用时间
- 流失率：客户停止付费的比例，流失率的倒数近似为客户平均生命周期
```

### 4.7 增长驱动因素

1. **IT支出上行周期**：企业数字化转型持续
2. **企业上云趋势**：从本地部署向云端迁移
3. **AI驱动产品升级**：AI功能带来产品差异化与附加值提升

## 六、电子行业研究框架

### 6.1 电子产业信息流转全链路

```
采集 → 传输 → 运算 → 存储 → 输出
│      │      │      │      │
传感器  有线/   处理器  存储器  显示面板
光学摄像 无线传输  (CPU)  (Flash)  OLED
音频接收        IoT芯片  SSD    Mini-LED
```

### 6.2 创新弹性公式

```
创新弹性 = 增量 × 渗透率 × 份额 × 价格 × 毛利率
        + Δ存量 × Δ份额 × Δ渗透率 × Δ价格 × Δ毛利率
```

### 6.3 量价关联机制

- **量增效应**：吸引玩家进入 → 竞争者增加 → 产能扩充 → 壁垒下降 → 价格下降
- **成本效应**：量增对单个企业降低成本 → 通过价格降低+工艺迭代完成市场出清
- **创新循环**：创新带来弹性 → 增量成为存量 → 存量竞争出清 → 格局稳定 → 下一轮创新

### 6.4 技术演进两大路径

| 路径 | 特征 | 案例 |
|------|------|------|
| **破坏性创新** | 从无到有，打造新赛道 | 手机之于电脑 |
| **升级性迭代** | 旧技术→新技术→新旧共存→淘汰旧技术 | 智能机之于功能机、第三代化合物半导体 |
