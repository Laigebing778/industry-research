# TMT行业研究框架

## 一、行业概述

**名称**：TMT行业研究智能体 (TMT Agent)

**定位**：通信、电子、计算机、传媒行业分析与投资研究助手

**核心能力**：
- 通信产业链分析
- 半导体产业链分析
- AI产业链分析
- 技术趋势判断
- 公司基本面分析
- 连接Choice API获取数据

## 二、行业分类

### 2.1 通信板块

| 类型 | 主要产品 | 驱动因素 |
|-----|---------|---------|
| **光模块** | 光收发模块 | 数据中心投资、AI算力 |
| **光通信** | 光纤光缆、光器件 | 5G建设、宽带升级 |
| **通信设备** | 基站、交换机 | 运营商Capex |
| **运营商** | 电信服务 | 用户数、ARPU |

### 2.2 电子板块

| 类型 | 主要产品 | 特点 |
|-----|---------|------|
| **半导体** | 芯片、设备、材料 | 国产替代主线 |
| **PCB** | 印刷电路板 | 消费电子+服务器 |
| **消费电子** | 手机、可穿戴 | 产品创新周期 |
| **面板** | 显示屏 | 周期性+技术升级 |

### 2.3 计算机板块

| 类型 | 主要产品 | 驱动因素 |
|-----|---------|---------|
| **AI算力** | 服务器、GPU | AI大模型训练 |
| **信创** | 国产软件、服务器 | 政策驱动 |
| **网络安全** | 安全产品 | 合规需求 |

## 五、价格影响因素

### 5.1 光模块定价

```
价格 = 成本 + 利润
     = 光芯片成本 + 封装成本 + 毛利

价格趋势：
- 新产品溢价高（800G/1.6T）
- 成熟产品价格下降（400G）
- 技术迭代驱动价值量提升
```

### 5.2 半导体定价

```
芯片价格 = 设计成本 + 制造成本 + 封测成本 + 利润

影响因素：
- 产能利用率（供需关系）
- 技术节点（先进/成熟）
- 竞争格局（国产替代进度）
```

### 5.3 周期性分析

| 子行业 | 周期性 | 周期长度 |
|-----|--------|---------|
| 存储芯片 | 强 | 3-4年 |
| 面板 | 强 | 2-3年 |
| PCB | 中 | 1-2年 |
| 光模块 | 弱 | 技术驱动 |
| 软件 | 弱 | 订阅模式 |

## 六、技术趋势分析

### 6.1 AI技术演进

```
大模型演进：
GPT-3(175B) → GPT-4(1.8T) → GPT-5(?)

算力需求：
训练：从万卡到十万卡
推理：成本下降，应用普及

关键趋势：
- 算力需求持续增长
- 端侧AI兴起
- 模型小型化
```

### 6.2 光通信技术

```
技术路线：
100G → 400G → 800G → 1.6T → 3.2T

新技术：
- CPO(光电共封装)
- LPO(线性直驱)
- 硅光技术
- 液冷散热
```

### 6.3 半导体技术

```
制程演进：
14nm → 7nm → 5nm → 3nm → 2nm

国产替代：
- 成熟制程(28nm+)：基本自给
- 先进制程(7nm)：突破中
- 设备材料：持续攻关
```

## 九、智能体功能设计

### 9.1 核心功能

1. **产业链分析**
   - 输入：子行业名称
   - 输出：产业链图谱、关键环节

2. **技术趋势**
   - 输入：技术类型
   - 输出：技术演进路线、投资机会

3. **供需分析**
   - 输入：产品类型
   - 输出：供需平衡、价格展望

4. **公司筛选**
   - 输入：排序指标
   - 输出：公司列表、排序结果

5. **估值分析**
   - 输入：公司代码
   - 输出：PE/PB历史分位、对比

6. **API查询生成**
   - 输入：数据类型
   - 输出：Choice API查询语句
