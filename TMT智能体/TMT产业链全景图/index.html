英伟达GB200配比关系（投资计算核心）

  • GPU:HBM = 1:2（每颗GPU配2颗HBM3e）→ HBM需求=GPU出货量×2
  • GPU:光模块 = 1:3（每颗GPU配3个光模块端口）→ 光模块需求=GPU出货量×3
  • GPU:光芯片 = 1:6（每个光模块2颗光芯片）→ 光芯片需求=GPU出货量×6
  • GPU:电源 = 2:1（每2颗GPU配1个电源模块）→ 电源需求=GPU出货量÷2
  • GPU:散热 = 1:1（每颗GPU配1套液冷）→ 散热需求=GPU出货量×1
  • GPU:PCB载板 = 1:1（每颗GPU配1块载板）→ PCB需求=GPU出货量×1

AI时代核心芯片专题 - 技术原理与投资标的

AI算力产业链中，芯片是最核心环节。本页详解四大类芯片：算力芯片、存储芯片、光芯片、交换芯片

一、算力芯片(GPU/NPU/ASIC) - 定价权最强

GPU架构原理

核心组件：
  • 计算核心(CUDA Core)：数千个并行计算单元，适合矩阵运算
  • Tensor Core：专门做AI矩阵乘法，A100有432个
  • HBM接口：连接高带宽内存，带宽>3TB/s
  • NVLink接口：GPU间高速互联，900GB/s
为什么GPU适合AI：
  • AI训练=大量矩阵乘法（并行计算）
  • GPU有数千核心vs CPU仅几十核心
  • 吞吐量：GPU比CPU快100-1000倍
  • HBM带宽：GPU 3TB/s vs CPU 50GB/s

  • HBM：192GB HBM3e

  • Tensor：AI矩阵加速

二、存储芯片(HBM/DRAM/NAND) - 定价权较强

HBM堆叠技术原理

HBM结构：
  • 垂直堆叠：8-12层DRAM芯片堆叠
  • TSV互联：穿透硅柱连接各层
  • CoWoS封装：与GPU封装在一起
  • 微凸点：每层间距<50μm
为什么需要HBM：
  • GPU算力太快，普通内存跟不上
  • HBM带宽3TB/s vs DDR5仅100GB/s
  • AI训练需要海量数据快速传输
  • GB200配比：GPU:HBM=1:2
HBM世代对比： HBM2e(256GB/s) → HBM3(600GB/s) → HBM3e(1024GB/s) → HBM4(研发中)

HBM堆叠结构示意图（8层HBM3e）

HBM3e关键参数：
  • 层数：8-12层堆叠
  • 带宽：1024GB/s
  • 容量：24GB/堆叠
  • 间距：<50μm/层
  • 单价：$20-30/GB

CoWoS封装技术原理

CoWoS是什么：
  • CoWoS = Chip-on-Wafer-on-Substrate
  • 台积电独家：2.5D先进封装技术
  • GPU+HBM：封装在同一硅中介层上
  • AI芯片必备：GB200/A100都用
封装流程：
  • 1. 硅中介层：承载GPU+HBM芯片
  • 2. TSV穿孔：连接上下层信号
  • 3. 微凸点：芯片与中介层焊接
  • 4. 封装基板：连接PCB电路板
国产封装现状： 长电科技/通富微电可做先进封装，但CoWoS技术台积电100%垄断。国产替代路径：看好封测厂技术突破。

三、光芯片(DFB/EML/VCSEL/硅光) - 定价权中等(国产有突破)

光芯片技术原理

光芯片作用：
  • 电信号→光信号：发射端(TOSA)
  • 光信号→电信号：接收端(ROSA)
  • 每个光模块需要2颗光芯片
  • GPU:光芯片配比=1:6
技术路线：
  • DFB：直接调制，成本低，10G/25G主流
  • EML：外调制，性能好，50G/100G用
  • VCSEL：垂直发光，短距离用
  • 硅光：集成度高，1.6T主流方案

光芯片结构示意图（EML激光器芯片）

  • EML：激光+调制集成
  • DFB：光栅选频单模
  • 波长：1310/1550nm
  • 速率：50-100Gbps
  • 国产：源杰科技突破
DFB vs EML：
DFB=激光器+光栅，直接调制
EML=DFB+EAM调制器分离，性能更好

四、光模块完整原理 - 800G/1.6T核心器件

光模块结构拆解

核心组件：
  • TOSA：光发射组件(激光器+驱动)
  • ROSA：光接收组件(探测器+放大)
  • DSP芯片：数字信号处理，Marvell/博通
  • PCB基板：电路载体
  • FAU/隔离器：光学辅助器件
价值拆解：
  • 光芯片：占BOM 30-40%
  • DSP芯片：占BOM 20-25%
  • 光器件：占BOM 15-20%
  • PCB/结构件：占BOM 10-15%
  • 800G光模块售价$600-800

800G光模块结构示意图

光模块关键参数：
  • TOSA：电→光转换

景气度跟踪方法论 - 行业周期判断核心

什么是景气度跟踪？

景气度是行业供需关系+盈利水平+预期变化的综合反映。景气度向上时，行业处于量价齐升→利润改善→估值提升的正反馈周期。
景气度≠股价：景气度领先股价1-3个季度，股价反映预期变化。
景气度跟踪目的：判断行业所处周期位置，预判拐点，选择投资时机。

景气度向上信号（买入时机）

  • 订单能见度延长：从3个月→6个月→12个月（需求强劲）
  • 产能利用率提升：从70%→85%→95%（供给紧张）
  • 产品价格上涨：ASP环比提升，涨价传导利润
  • 库存主动去化：库存周转加快，渠道库存下降
  • Capex扩张：扩产计划出台，行业信心增强
  • 业绩前瞻上调：公司上调业绩指引，超预期

景气度向下信号（卖出时机）

  • 订单能见度缩短：从12个月→6个月→3个月（需求疲软）
  • 产能利用率下降：从95%→85%→70%（供给过剩）
  • 产品价格下跌：ASP环比下降，降价压缩利润
  • 库存被动累积：库存周转放缓，渠道库存上升
  • Capex收缩：扩产计划推迟，行业信心下降
  • 业绩前瞻下调：公司下调业绩指引，低于预期

景气度周期四阶段

阶段1：复苏
订单回升、库存去化、价格企稳
阶段2：扩张
量价齐升、产能紧张、利润改善
阶段3：放缓
订单放缓、库存累积、价格下跌
阶段4：衰退
需求疲软、产能过剩、利润下滑
