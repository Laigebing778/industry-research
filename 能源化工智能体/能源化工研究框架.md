## 三、产业链结构

### 3.1 石化产业链

```
上游          中游              下游
原油开采  →  炼油  →  烯烃  →  塑料  →  终端
   ↓         ↓         ↓         ↓
中国石油   中国石化   恒力石化   金发科技
中国海油   荣盛石化   东方盛虹   国恩股份
          恒逸石化   卫星化学
```

**价格传导**：
```
原油价格 → 石脑油价格 → 烯烃价格 → 塑料价格
        (正相关)      (相关+加工费) (相关+需求)
```

### 3.2 烯烃产业链

```
原料路线：
油制烯烃：石脑油 → 蒸汽裂解 → 乙烯/丙烯
煤制烯烃：煤 → 甲醇 → MTO → 乙烯/丙烯
轻烃路线：乙烷/丙烷 → 裂解 → 乙烯/丙烯

下游产品：
乙烯 → 聚乙烯(PE)、乙二醇、PVC
丙烯 → 聚丙烯(PP)、环氧丙烷、丙烯酸
```

**成本对比**：
| 路线 | 成本优势 | 适用地区 |
|-----|---------|---------|
| 油制 | 成熟稳定 | 沿海炼厂 |
| 煤制 | 油价>60美元有优势 | 西部煤产区 |
| 轻烃 | 成本最低 | 美国、中东 |

### 3.3 化纤产业链

```
原油 → 石脑油 → PX → PTA → 聚酯 → 涤纶
                  ↓
              乙二醇(EG)
```

**主要玩家**：
- 桐昆股份：PTA-聚酯龙头
- 恒逸石化：炼化-化纤一体化
- 新凤鸣：涤纶长丝
- 东方盛虹：炼化+化纤

### 3.4 煤化工产业链

```
煤 → 煤气化 → 合成气 → 甲醇 → MTO → 烯烃
         ↓
      合成氨 → 尿素
         ↓
      乙二醇
```

**中国特色**：
- 富煤贫油少气
- 煤制烯烃成本低
- 双碳政策约束新增产能

```
油价上涨时：
原油 ↑ → 石脑油 ↑ → 烯烃 ↑ → 塑料 ↑

油价下跌时：
原油 ↓ → 石脑油 ↓ → 烯烃 ↓ → 塑料 ↓
       但加工利润可能改善
```

## 五、价格影响因素

### 5.1 油价驱动框架

```
供给端                需求端
─────────────────────────────────────────────
OPEC+产量    ←→    全球经济增长
美国页岩油          中国需求
俄罗斯供应          季节性因素
地缘政治            库存水平
```

**关键价格区间**：
- <50美元：页岩油减产
- 50-70美元：OPEC+减产保价
- 70-90美元：均衡区间
- >90美元：需求抑制

## 六、周期分析

### 6.1 化工周期特征

```
周期阶段        行业特征              投资策略
─────────────────────────────────────────────
复苏期    需求改善，价格底部回升    买入龙头
繁荣期    产能紧张，利润高企        持有观察
衰退期    产能过剩，价格下跌        规避风险
萧条期    产能出清，价格触底        左侧布局
```

### 6.2 库存周期

| 阶段 | 价格 | 库存 | 利润 | 策略 |
|-----|------|------|------|------|
| 被动补库 | ↑ | ↑ | 高 | 持有 |
| 主动去库 | ↓ | ↓ | 低 | 观望 |
| 被动去库 | ↑ | ↓ | 回升 | 买入 |
| 主动补库 | ↑ | ↑ | 高位 | 减持 |
