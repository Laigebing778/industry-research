## 二、产业链结构

### 2.1 石油产业链结构

```
上游（原油供给）          中游（炼化加工）          下游（化工产品）
────────────────────────────────────────────────────────────────────
OPEC国家                  一次加工                  乙烯产业链
├─ 沙特、阿联酋           ├─ 常压蒸馏               └─ 石脑油→乙烯→LLDPE、丁二烯、纯苯
├─ 科威特、伊拉克         └─ 减压蒸馏
                          二次加工                  芳烃产业链
非OPEC国家                ├─ 热裂化                 └─ 原油→石脑油→PX→PTA→聚酯
├─ 美国页岩油             ├─ 催化裂化
├─ 俄罗斯                 ├─ 延迟焦化
├─ 巴西、圭亚那           ├─ 催化重整
└─ 加拿大                 └─ 加氢裂化
                          三次加工
海上油气                  └─ 烷基化、异构化
└─ 深海项目（成本约40美金/桶）
```

### 2.2 炼化一体化优势

**一体化率公式**：
```
炼化一体化率 = (乙烯原料 + 丙烷 + 副产丙烯量 + 重整原料数量) / 原油加工能力
```

| 对比项 | 民营大炼化 | 全国平均 |
|-------|-----------|---------|
| 一体化率 | 62% | 10% |
| 化工品比例 | 高 | 低 |
| 成品油收率 | 低 | 高 |
| 单位投资 | 低 | 高 |
| 单位成本 | 低 | 高 |

**核心优势**：
- 物料互供、公用工程共享
- 化工品比例最大化，减少成品油收率
- 单位投资、单位成本更低

### 2.3 全球炼厂竞争力评价

| 指标 | 说明 | 全球对比 |
|------|------|---------|
| **Nelson复杂指数(NCI)** | 衡量炼厂加工深度，指数越高，高价值产品能力越强 | 北美11.6 > 亚太9.6 > 欧洲9.2 > 中东6.9 |
| **FCC转换率** | 催化裂化转换能力 | 北美71% > 亚太64% > 欧洲42% > 中东27% |
| **化工品比例** | 化工品产出占比 | 一体化炼厂更高 |

### 4.4 化工品价差分析（核心）

**关键公式**：
```
价差 = 销售价格 - 原料成本 - 其他加工成本
```

**重要原则**：
1. **避免价格陷阱**：成本推涨价差不扩大的价格上涨对企业盈利贡献较小
2. **持续性判断**：从供需关系分析当前景气周期的持续性
3. **景气位置判断**：从历史相对分位看当前景气位置
4. **弹性判断**：根据相关公司盈利弹性选择标的

**关键价差指标**：

| 产品 | 价差指标 | 正常区间 |
|-----|---------|---------|
| 乙烯 | 乙烯-石脑油 | 300-500美元/吨 |
| PX | PX-石脑油 | 300-400美元/吨 |
| PTA | PTA-PX | 100-200美元/吨 |
| 涤纶 | 涤纶-PTA-EG | 800-1200元/吨 |

## 五、库存周期分析

### 5.1 石化行业周期规律

| 周期 | 时间段 | 驱动因素 |
|------|--------|----------|
| 第一轮 | 2001-2008年 | 中国加入WTO带动 |
| 第二轮 | 2009-2015年 | 四万亿投资带动 |
| 第三轮 | 2016-2022年 | 供给侧改革带动 |
| 第四轮 | 2023年起 | 海外产能退出，国内龙头竞争力增强 |

**周期长度**：约7年一个周期

### 5.2 美国库存井(DUC)周期

- 2024年4月DUC降至4510台（历史低位）
- 以消耗库存井增产的模式难以持续
- 页岩油预期2026年左右达峰

### 5.3 涤纶长丝库存周期

- 长丝库存处于历史相对低位
- 2024年产能增速明显放缓（仅1%）
- 海外补库预期+内需韧性→景气上行

### 5.4 OECD库存指标

| 时间 | 库存天数 |
|------|----------|
| 正常水平 | 60-62天 |
| 2021年 | 64.1天 |
| 2023年 | 61.3天 |
| 2025E | 62.0天 |

### 5.5 库存周期四阶段

| 阶段 | 价格 | 库存 | 利润 | 策略 |
|-----|------|------|------|------|
| 被动补库 | ↑ | ↑ | 高 | 持有 |
| 主动去库 | ↓ | ↓ | 低 | 观望 |
| 被动去库 | ↑ | ↓ | 回升 | 买入 |
| 主动补库 | ↑ | ↑ | 高位 | 减持 |
