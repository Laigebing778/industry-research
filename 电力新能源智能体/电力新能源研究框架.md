## 二、行业分类

### 2.1 电力板块

| 类型 | 特点 | 驱动因素 |
|-----|------|---------|
| **火电** | 基荷电源，调峰 | 煤价、电价、利用小时 |
| **水电** | 清洁能源，成本低 | 来水、装机容量 |
| **核电** | 基荷电源，稳定 | 审批节奏、利用小时 |
| **绿电** | 新能源发电 | 装机增长、补贴政策 |

### 2.2 新能源板块

| 类型 | 成熟度 | 主要玩家 |
|-----|--------|---------|
| **光伏** | 成熟期，成本持续下降 | 中国主导全产业链 |
| **风电** | 成熟期，大型化趋势 | 整机厂+零部件 |
| **储能** | 成长期，政策驱动 | 电池厂+系统集成商 |
| **电网设备** | 稳定期，智能化升级 | 国网系+民企 |

## 三、产业链结构

### 3.1 光伏产业链

```
上游          中游              下游
多晶硅  →  硅片  →  电池片  →  组件  →  电站
   ↓         ↓         ↓         ↓
通威股份   隆基绿能   通威股份   天合光能
大全能源   TCL中环   爱旭股份   晶科能源
协鑫科技   弘元绿能   钧达股份   晶澳科技
```

**关键环节成本**：
- 多晶硅：成本约40-60元/kg
- 硅片：成本约1.5-2元/片
- 电池片：成本约0.8-1元/W
- 组件：成本约1.2-1.5元/W

### 3.2 风电产业链

```
上游          中游              下游
零部件  →  整机  →  风场建设  →  运营
   ↓         ↓
叶片       金风科技
塔筒       明阳智能
轴承       运达股份
齿轮箱     电气风电
电缆
```

**大型化趋势**：
- 陆风：从2MW向5-6MW升级
- 海风：从4MW向10-15MW升级

### 3.3 储能产业链

```
上游          中游              下游
电池材料  →  电芯  →  系统集成  →  应用
   ↓         ↓         ↓
锂矿       宁德时代   阳光电源
正极材料   比亚迪     华为数字能源
电解液     亿纬锂能   科华数据
隔膜       国轩高科   上能电气
```

**应用场景**：
- 电源侧：配储强制，新能源配储10-20%
- 电网侧：独立储能电站
- 用户侧：工商业储能、户储

### 3.4 电网设备产业链

```
一次设备              二次设备
─────────────────────────────────────────────
变压器        保护设备
开关          监控系统
电缆          调度系统
绝缘子        智能终端
```

**投资驱动**：
- 特高压建设
- 配电网改造
- 智能化升级
- 新型电力系统

## 四、供需平衡分析

### 4.1 电力供需平衡

```
供给端              需求端
─────────────────────────────────────────────
火电装机    +    工业用电
水电装机    +    居民用电
```

## 五、价格影响因素

### 5.1 组件价格驱动

```
长期因素           中期因素          短期因素
─────────────────────────────────────────────────────
技术进步    ←→    产能投放    ←→    价格博弈
规模效应          政策补贴            库存变化
原材料成本        季节性因素          市场情绪
```

### 5.2 电价影响因素

| 类型 | 定价机制 | 影响因素 |
|-----|---------|---------|
| 火电 | 市场化交易 | 煤价、供需 |
| 水电 | 核价+市场 | 来水、政策 |
| 核电 | 核价为主 | 成本 |
| 绿电 | 市场化 | 补贴、绿证 |

### 5.3 储能经济性

```
储能收益 = 峰谷价差收益 + 辅助服务收益 + 容量租赁收益

成本 = 初始投资 + 运维成本 + 更换成本

关键变量：
- 峰谷价差（>0.7元/kWh有经济性）
- 循环次数（>6000次）
- 系统成本（<1.5元/Wh）
```

## 六、政策影响分析

### 6.1 双碳目标

```
2025年：非化石能源消费比重20%
2030年：碳达峰，非化石能源25%
2060年：碳中和，非化石能源80%+
```

### 6.2 关键政策

| 政策 | 影响 |
|-----|------|
| 新能源配储 | 强制配储10-20%，2-4小时 |
| 分时电价 | 峰谷价差拉大，利好储能 |
| 绿电交易 | 绿证价值体现 |
| 电力市场化 | 火电盈利改善 |

### 6.3 新型电力系统

```
传统电力系统        新型电力系统
─────────────────────────────────────────────
源随荷动    →    源网荷储互动
集中式为主  →    集中式+分布式
火电为主    →    新能源为主
单向潮流    →    双向潮流
```
