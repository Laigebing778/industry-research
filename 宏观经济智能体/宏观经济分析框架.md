## 二、宏观经济分析框架

### 2.1 经济增长分析

#### GDP分析
```
GDP = 消费 + 投资 + 政府支出 + 净出口
    = C + I + G + (X-M)
```

| 分项 | 权重 | 关注指标 | 数据来源 |
|-----|------|---------|---------|
| 消费 | ~55% | 社零总额、居民收入、消费信心指数 | 国家统计局 |
| 投资 | ~40% | 固定资产投资、房地产投资、制造业投资 | 国家统计局 |
| 净出口 | ~5% | 进出口金额、贸易顺差 | 海关总署 |

#### 生产端分析
| 指标 | 含义 | 发布频率 |
|-----|------|---------|
| 工业增加值 | 工业生产景气度 | 月度 |
| PMI | 制造业景气度（荣枯线50） | 月度 |
| 发电量 | 经济活动强度 | 日度/月度 |

---

### 2.2 通胀分析

#### CPI（消费者物价指数）
```
CPI = 食品 + 非食品
    = 猪肉 + 蔬菜 + ... + 核心CPI
```

| 分项 | 权重 | 特点 |
|-----|------|------|
| 食品 | ~30% | 波动大，受季节和供给影响 |
| 非食品 | ~70% | 相对稳定，反映需求 |
| 核心CPI | - | 剔除食品能源，反映真实需求 |

#### PPI（生产者物价指数）
- 反映工业企业出厂价格
- 领先CPI约3-6个月
- 与大宗商品价格高度相关

#### 通胀传导链条
```
大宗商品价格 → PPI → 企业利润 → CPI → 居民生活成本
```

### 2.3 货币与流动性

#### 货币供应量
| 指标 | 定义 | 含义 |
|-----|------|------|
| M0 | 流通中现金 | 最活跃的货币 |
| M1 | M0 + 企业活期存款 | 现实购买力 |
| M2 | M1 + 定期存款 + 储蓄存款 | 潜在购买力 |

#### 关键利差
```
M1-M2剪刀差：反映企业活期化程度，预示经济活力
M2-社融剪刀差：反映金融体系资金空转程度
```

#### 利率体系
| 利率 | 含义 | 影响 |
|-----|------|------|
| OMO利率 | 央行短期政策利率 | 资金面松紧 |
| MLF利率 | 中期政策利率 | LPR定价基准 |
| LPR | 贷款市场报价利率 | 实体融资成本 |
| 10Y国债 | 无风险利率 | 资产定价锚 |

---

### 2.4 财政政策

#### 财政收支
```
财政赤字 = 财政支出 - 财政收入
赤字率 = 财政赤字 / GDP
```

#### 财政工具
| 工具 | 作用 | 特点 |
|-----|------|------|
| 减税降费 | 增加企业利润 | 间接刺激 |
| 专项债 | 基建投资 | 直接拉动 |
| 转移支付 | 支持地方 | 平衡区域 |

#### 财政发力观察
- 专项债发行进度
- 基建投资增速
- 政府性基金收支

---

### 2.5 汇率与国际收支

#### 汇率影响因素
```
汇率 = 利差 + 贸易顺差 + 资本流动 + 政策干预
```

#### 关键指标
| 指标 | 含义 |
|-----|------|
| 美元指数 | 美元相对一篮子货币强弱 |
| 中美利差 | 10Y国债利差，影响资本流动 |
| 外汇储备 | 央行干预能力 |
| 结售汇差额 | 资本流动方向 |

---

### 2.6 房地产市场

#### 房地产周期
```
政策 → 销售 → 资金 → 开工 → 投资
```

#### 关键指标
| 指标 | 含义 | 领先性 |
|-----|------|--------|
| 销售面积 | 市场需求 | 领先3-6月 |
| 新开工面积 | 开发商信心 | 同步 |
| 开发投资 | 经济贡献 | 滞后 |
| 土地成交 | 政府收入 | 领先 |

## 三、经济周期判断

### 3.1 美林时钟

| 阶段 | 经济 | 通胀 | 最佳资产 |
|-----|------|------|---------|
| 复苏 | ↑ | ↓ | 股票 |
| 过热 | ↑ | ↑ | 商品 |
| 滞胀 | ↓ | ↑ | 现金 |
| 衰退 | ↓ | ↓ | 债券 |

### 3.2 中国特色周期

```
政策周期 → 金融周期 → 经济周期 → 盈利周期
```

## 五、智能体功能设计

### 5.1 核心功能

1. **数据解读**
   - 输入：经济数据
   - 输出：数据含义、历史对比、市场预期

2. **周期判断**
   - 输入：多维度经济指标

3. **政策分析**

   - 输出：政策意图、影响路径

4. **资产配置建议**
   - 输入：经济周期判断
   - 输出：大类资产配置建议
