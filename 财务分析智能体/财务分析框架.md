## 二、财务报表分析框架

### 2.1 三大报表关系

```
资产负债表（存量）  ←→  利润表（流量）  ←→  现金流量表（流量）
      ↓                    ↓                    ↓
   资产结构            盈利能力             现金质量
   资本结构            利润来源             收现能力
```

**会计方程**：资产 = 负债 + 所有者权益

---

### 2.2 资产负债表解读

#### 资产端
| 科目 | 含义 | 关注点 |
|-----|------|--------|
| 现金 | 可用于支付的银行余额 | 流动性、现金充足性 |
| 应收账款 | 待收客户已供货款项 | 客户质量、回款能力 |
| 存货 | 待出售商品 | 周转效率、跌价风险 |
| 预付账款 | 已支付供应商待供货款 | 供应商议价能力 |
| 固定资产 | 长期经营资产 | 产能、折旧政策 |

#### 负债端
| 科目 | 含义 | 关注点 |
|-----|------|--------|
| 应付账款 | 待付供应商货款 | 供应商议价能力 |
| 预收账款 | 已收客户待供货款项 | 客户预付意愿 |
| 短期借款 | 一年内待还借款 | 短期偿债压力 |
| 长期借款 | 一年后待还借款 | 长期资本结构 |

#### 所有者权益
| 科目 | 含义 |
|-----|------|
| 股本 | 股东投入本金 |
| 留存利润 | 未分配利润 |

---

### 2.3 利润表解读

```
收入（规模与结构）
  - 成本（毛利率 = 1 - 成本/收入）
  = 毛利
  - 费用（费用率 = 费用/收入）
  = 营业利润（自身经营能力）
  + 投资收益（子公司贡献）
  - 利息（融资成本）
  - 所得税（政府分成）
  = 净利润（股东所得）
```

#### 关键分析点
| 项目 | 分析要点 |
|-----|---------|
| 收入 | 规模、结构、增长率、市场份额 |
| 成本 | 毛利率、成本控制能力 |
| 费用 | 费用率、管理效率 |
| 营业利润 | 核心盈利能力 |
| 投资收益 | 子公司贡献、投资质量 |
| 利息 | 融资成本、财务杠杆 |
| 净利润 | 最终盈利、股东回报 |

---

### 2.4 现金流量表解读

#### 净利润与经营现金流调节

```
净利润
+ 折旧摊销等非现金支出
- 应收款项增加（占用现金）
- 预付账款增加（占用现金）
- 存货增加（占用现金）
+ 预收账款增加（收到现金）
+ 应付款项增加（占用供应商资金）
= 经营净现金流
```

#### 现金流质量判断

| 净利润 | 经营现金流 | 含义 |
|-------|-----------|------|
| + | + | 优质企业 |
| + | - | 利润质量差，需关注应收/存货 |
| - | + | 亏损但有现金流入，可能扭亏 |
| - | - | 危险状态 |

## 三、财务指标体系

### 3.1 盈利能力指标

| 指标 | 公式 | 含义 |
|-----|------|------|
| ROE | 净利润/净资产 | 股东回报率 |
| ROA | 净利润/总资产 | 资产回报率 |
| ROCE | 息税前利润/占用资本 | 资本使用效率 |
| 销售净利率 | 净利润/营业收入 | 每元收入创造利润 |
| 销售毛利率 | 毛利/营业收入 | 产品定价能力 |
| 营业利润率 | 营业利润/营业收入 | 核心经营盈利能力 |

### 3.2 营运能力指标

| 指标 | 公式 | 含义 |
|-----|------|------|
| 应收账款周转率 | 营业收入/平均应收账款 | 回款速度 |
| 存货周转率 | 营业成本/平均存货 | 销售速度 |
| 应付账款周转率 | 营业成本/平均应付账款 | 付款速度 |
| 固定资产周转率 | 营业收入/平均固定资产 | 资产效率 |
| 总资产周转率 | 营业收入/平均总资产 | 资产运营效率 |

### 3.3 偿债能力指标

| 指标 | 公式 | 标准 | 含义 |
|-----|------|------|------|
| 资产负债率 | 负债/资产 | <70% | 财务风险 |
| 流动比率 | 流动资产/流动负债 | >1.5 | 短期偿债能力 |
| 速动比率 | 速动资产/流动负债 | >1 | 快速变现能力 |
| 利息保障倍数 | EBIT/利息支出 | >3 | 利息偿还能力 |

### 3.4 成长性指标

| 指标 | 公式 | 含义 |
|-----|------|------|
| 营收增长率 | (本期收入-上期收入)/上期收入 | 收入成长 |
| 净利润增长率 | (本期利润-上期利润)/上期利润 | 利润成长 |
| 总资产增长率 | (本期资产-上期资产)/上期资产 | 规模扩张 |

## 四、杜邦分析体系

### 4.1 杜邦分解

```
ROE = 净利润/净资产

    = (净利润/销售收入) × (销售收入/总资产) × (总资产/净资产)

    = 销售净利率 × 资产周转率 × 权益乘数

    = 利润率 × 效率 × 杠杆
```

### 4.2 三因子分解

| 因子 | 公式 | 代表能力 | 提升途径 |
|-----|------|---------|---------|
| 销售净利率 | 净利润/收入 | 盈利能力 | 产品定价、成本控制 |
| 资产周转率 | 收入/总资产 | 运营效率 | 存货管理、应收管理 |
| 权益乘数 | 总资产/净资产 | 财务杠杆 | 负债经营 |

### 4.3 企业战略映射

| 高利润率 | 低利润率 |
|---------|---------|
| **低周转率** | 产品领先型企业 | **高周转率** | 流程卓越型企业 |
| 技术垄断、品牌溢价 | 成本控制、高效运营 |

## 五、财务舞弊识别

### 5.1 舞弊动机
- 融资需求（贷款、IPO）
- 股价维护
- 业绩对赌
- 管理层利益

### 5.2 舞弊进化路径

| 版本 | 手法 | 特征 | 识别要点 |
|-----|------|------|---------|
| **1.0** | 虚增收入→应收账款 | 应收激增 | 应收周转率下降、账龄恶化 |
| **2.0** | 虚增利润→其他流动资产 | 存货/预付激增 | 存货周转率下降、预付异常 |
| **3.0** | 虚增利润→长期资产 | 长期投资激增 | 固定资产周转率下降 |
| **4.0** | 体外循环提供现金流 | 现金流匹配 | 关联交易、客户异常 |
| **5.0** | 权力寻租 | 审计合谋 | 审计意见、监管处罚 |

### 5.3 舞弊预警信号

| 信号类型 | 具体表现 |
|---------|---------|
| **利润与现金流背离** | 净利润持续为正，经营现金流持续为负 |
| **应收账款异常** | 应收增速 > 收入增速，账龄恶化 |
| **存货异常** | 存货增速 > 收入增速，存货周转率下降 |
| **其他应收款异常** | 大额其他应收款，关联方占用 |
| **毛利率异常** | 毛利率远高于同行，或持续上升 |
| **关联交易频繁** | 大量关联交易，定价不透明 |

## 六、企业估值框架

### 6.1 估值方法概览

| 方法 | 适用场景 | 核心逻辑 |
|-----|---------|---------|
| DCF | 成熟稳定企业 | 未来现金流折现 |
| PE | 盈利稳定企业 | 市盈率×净利润 |
| PB | 重资产企业 | 市净率×净资产 |
| PS | 亏损/高增长企业 | 市销率×收入 |
| EV/EBITDA | 资本密集型企业 | 企业价值/息税折旧前利润 |

### 6.2 DCF估值模型

```
企业价值 = Σ(FCFt / (1+WACC)^t) + TV / (1+WACC)^n

其中：
FCF = 自由现金流 = EBIT × (1-T) + 折旧摊销 - 资本支出 - 营运资本变动
WACC = 加权平均资本成本
TV = 终值 = FCFn+1 / (WACC - g)
g = 永续增长率
```

### 6.3 相对估值法

#### PE估值
```
股价 = PE × EPS
PE = 股价 / 每股收益
```

- 适用于盈利稳定的企业
- PE受增长率、风险、利率影响
- 行业PE作为参照

#### PB估值
```
股价 = PB × BPS
PB = 股价 / 每股净资产
```

- 适用于重资产行业
- 关注ROE与PB的关系
- PB < 1 可能被低估

## 七、智能体功能设计

### 7.1 核心功能

1. **财务指标计算**
   - 输入：财务报表数据
   - 输出：各项财务指标

2. **杜邦分析**
   - 输入：利润表、资产负债表
   - 输出：ROE分解、因子贡献

3. **财务舞弊预警**
   - 输入：财务报表数据
   - 输出：舞弊风险评分、预警信号

4. **企业估值**
   - 输入：财务数据、估值参数
   - 输出：估值结果

### 7.2 分析框架

```
输入财务数据
    ↓
计算财务指标
    ↓
杜邦分析分解
    ↓
同行对比定位
    ↓
舞弊风险筛查
    ↓
估值模型计算
    ↓
输出分析报告
```
